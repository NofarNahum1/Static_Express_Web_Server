Q: How to check if Node.js is already installed on my PC?

A: To check if Node.js is already installed on your PC, follow these steps:
     Open Command Prompt or Terminal:
     On Windows, you can open Command Prompt by pressing Win + R, typing cmd, and hitting Enter.
     Check Node.js Version:
     In the Command Prompt or Terminal, type the following command and press Enter:
     node -v
     This command will output the version number of Node.js if it is installed

Q: What is a static express web server?

A: A static Express web server is a server that serves static files, such as HTML, CSS, JavaScript, images, and other assets,
    using the Express.js framework in a Node.js environment.
    Static files are those that do not change in response to user input or other dynamic factors.
